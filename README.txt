## Objetivo
===========================================================================================================
Esse programa foi desenvolvido junto ao curso de Algoritmos e Programação da UFRGS com o intuito de contribuir para o aprendizado do conteúdo


## Descrição
===========================================================================================================
* Este é um jogo feito com a biblioteca gráfica Raylib, inspirado no Flappy Bird. O objetivo é que o jogador navegue por obstáculos sem colidir com eles enquanto desce. Cada obstáculo ultrapassado incrementa o placar em 50 pontos.

* Pressione a barra de espaço para fazer o personagem do jogador subir. Pressione "esc" para sair do jogo.

* O menu do jogo oferece opções de dificuldade (fácil, médio, difícil), um placar mostrando os 5 maiores pontuações e o nome do jogador.

* O programa inclui assets como sprites e arquivos de áudio.


## Recursos
===========================================================================================================
* Jogabilidade inspirada no Flappy Bird
* Rastreamento de pontuação
* Leitura de níveis de dificuldade
* Placar com nomes dos jogadores
* Entrada do usuário para pontuações altas


## Requisitos
===========================================================================================================
O jogo foi desenvolvido e rodado usando Windows 10


## Controles
===========================================================================================================
* Barra de espaço: Subir
* Esc: Sair do jogo


## Placar
===========================================================================================================
O placar consiste em 5 entradas (nome e score) que são armazenados em um arquivo texto. Se o jogador alcançar uma pontuação alta, o jogo solicita que ele insira seu nome.
